API

Una API (Interfaz de Programación de Aplicaciones) es un conjunto de características y reglas que existen dentro de un programa de software (la aplicación) y que permiten la interacción con él a través del software, a diferencia de una interfaz de usuario humana. La API puede considerarse un simple contrato (la interfaz) entre la aplicación que la ofrece y otros elementos, como software o hardware de terceros.

En el desarrollo web, una API es generalmente un conjunto de características de código (por ejemplo, métodos , propiedades , eventos y URL ) que un desarrollador puede usar en sus aplicaciones para interactuar con componentes del navegador web de un usuario, otro software/hardware en la computadora del usuario o sitios web y servicios de terceros. https://developer.mozilla.org/en-US/docs/Glossary/API

¿Características de api en el web?

una API (Application Programming Interface) en el contexto web es un conjunto de reglas, protocolos y herramientas que permite que diferentes aplicaciones de software se comuniquen entre sí. Piensa en ella como un "menú" o "contrato" que especifica cómo un programa puede solicitar servicios o datos de otro. Sus caracteristicas son Interoperabilidad, Abstracción, Modularidad y Reusabilidad, Escalabilidad, Seguridad, Documentación, Versionamiento, Statelessness (en RESTful APIs).

¿Para qué sirven?

Las APIs son el motor de la web moderna. Sirven para: Integración de Servicios, Creación de Ecosistemas, esacoplamiento de Arquitecturas, Enriquecimiento de la Experiencia del Usuario, Automatización de Tareas, Desarrollo Móvil y Multiplataforma.

¿Requisitos para su utilización?

Para utilizar (o consumir) una API web, generalmente se necesitan los siguientes elementos: Conocimiento de HTTP/HTTPS, Cliente HTTP, URL (Endpoint) de la API, Parámetros de Solicitud, Credenciales de Autenticación/Autorización, Formato de Datos, Documentación de la API.

¿Ejemplos de Uso?

Las APIs están en todas partes en la web. Aquí tienes algunos ejemplos comunes: APIs de Redes Sociales (Facebook Graph API, Twitter API), APIs de Mapas y Localización (Google Maps API, OpenStreetMap API), APIs de Pagos (Stripe API, PayPal API), APIs Meteorológicas (OpenWeatherMap API, AccuWeather API), APIs de Información (Wikipedia API, Star Wars API - SWAPI), APIs Internas (Microservicios).
